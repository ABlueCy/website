The following `vtctl` commands are available for administering Cell Aliases.

## Commands

### AddCellsAlias

```
AddCellsAlias  [-cells <cell,cell2...>] <alias>
```

### UpdateCellsAlias

```
UpdateCellsAlias  [-cells <cell,cell2,...>] <alias>
```

### DeleteCellsAlias

```
DeleteCellsAlias  <alias>
```

### GetCellsAliases

```
GetCellsAliases
```
