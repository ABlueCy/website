The following `vtctl` commands are available for administering Cells.

## Commands

### AddCellInfo

```
AddCellInfo  [-server_address <addr>] [-root <root>] <cell>
```

### UpdateCellInfo

```
UpdateCellInfo  [-server_address <addr>] [-root <root>] <cell>
```

### DeleteCellInfo

```
DeleteCellInfo  [-force] <cell>
```

### GetCellInfoNames

```
GetCellInfoNames
```

### GetCellInfo

```
GetCellInfo  <cell>
```
