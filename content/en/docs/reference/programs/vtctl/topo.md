The following `vtctl` commands are available for administering Topology Services.

## Commands

### TopoCat

```
TopoCat  [-cell <cell>] [-decode_proto] [-decode_proto_json] [-long] <path> [<path>...]
```

### TopoCp

```
TopoCp  [-cell <cell>] [-to_topo] <src> <dst>
```
