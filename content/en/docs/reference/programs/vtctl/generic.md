The following generic `vtctl` commands are available for administering Vitess.

## Commands

### Validate

```
Validate [-ping-tablets]
```

### ListAllTablets

```
ListAllTablets <cell name1>, <cell name2>, ...
```

### ListTablets

```
ListTablets <tablet alias> ...
```

### Help

```
Help [command name]
```
